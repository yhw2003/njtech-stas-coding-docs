# Linux
[Source Code](https://www.kernel.org/)

linux是一个开源的，维护积极的，操作系统内核，提供了丰富的系统功能，广泛的硬件兼容，和生产级别的稳定性。
## Shell
shell是一套基于文本的与操作系统交互的逻辑。
对于初次接触shell的人而言，最大的困惑就是，这个东西它能干啥？它怎么做？与UI界面相比较，shell上空荡荡的啥都没有。我们必须凭借记忆和经验来判断下一步该怎么做。
